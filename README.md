## How to use this project locally
1. Extract the project files.
2. Open `index.html` in your browser to view the site locally.
3. To set up a GitHub repo and deploy with GitHub Pages, follow the steps below.

## Git & GitHub setup (commands to run locally)
```bash
# create a new repo on GitHub (name it campus-life-super-app or similar)
git init
git add .
git commit -m "initial MVP commit - basic pages and assets"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git push -u origin main

# do further commits as you make progress:
git add .
git commit -m "added map and sports pages, connected JS and CSS"
git push

# When ready, enable GitHub Pages in your repo settings (branch: main / folder: root)
# Your site will be available at: https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/
```
